**Hisaab.AI** is not just another expense tracker. It is a highly sophisticated, AI-driven personal finance assistant wrapped in an ultra-premium, dark-mode focused glassmorphic UI. 

Built with the modern user in mind, Hisaab.AI allows you to log transactions, split bills, and track your overall net worth entirely **hands-free using Hinglish Voice Commands**. Just speak to **"Bro-Bot"**, our proprietary NLP engine, and it will handle the rest.

## 🔥 Mind-Blowing Features

### 🎙️ 1. "Bro-Bot" AI & Smart Hands-Free Engine
*   **Hinglish NLP Support**: Speak naturally! Command the bot by saying, *"Rahul ko 500 rent aur 200 petrol ke diye"* or *"Aaj salary aai 50000"*.
*   **Multi-Entry Parsing**: Bro-Bot can slice and dice complex sentences. If you say, *"100 Pizza 200 Rupees Burger"*, it automatically drops the noise and creates two separate entries!
*   **Smart Silence Detection**: You don't even need to press "Stop" or "Enter". Speak, pause for 1.8 seconds, and the app automatically processes your command with visual pulsing feedback.

### 🎨 2. Ultra-Premium Glassmorphic Design
*   **Sleek Dark Mode**: Designed for OLED screens with rich indigo, violet, and pitch-black aesthetics.
*   **Micro-Interactions**: Smooth Framer-Motion animations, glowing borders, and reactive components that make the UI feel alive.
*   **Precision Dashboard**: Your "Net Liquidity" (Bank Balance), "Total In" (All Deposits), and "Total Out" (Expenses & Lent) are calculated and displayed in an easy-to-read, card-based layout.

### 🤝 3. Dual-Sync Lending & Split Engine
*   **Ledger + Dashboard Sync**: When you lend money to a friend, not only is their tab updated, but your main Cash Outflow is automatically adjusted.
*   **Automated Net Balances**: Automatically calculates who owes whom. If you paid ₹500 today and they paid ₹200 tomorrow, the system shows exactly who owes ₹300.
*   **Debt Clearing**: Settle accounts with a single click.

### ☁️ 4. Vercel Postgres Cloud Architecture
*   **Zero Data Loss**: Upgraded from local SQLite to a robust PostgreSQL database hosted on Vercel.
*   **True Cross-Platform**: Your data syncs instantly whether you are using the web version on your laptop or the native APK on your Android phone.

## 🛠️ The Tech Stack

This project leverages a modern, high-performance tech stack:

*   **Frontend**: React 19, Tailwind CSS 3.4, Framer Motion, Lucide React, Vite.
*   **Backend**: Flask 3.1.0, SQLAlchemy (ORM).
*   **Database**: Vercel Serverless Postgres / SQLite (for local dev dev).
*   **Mobile Wrapper**: Capacitor JS.
*   **AI Speech**: Web Speech API (en-IN localization).

## 🚀 Run It Locally (For Devs)

Want to see the magic under the hood?

### Prerequisites
*   Node.js (v18+)
*   Python 3.10+
*   PostgreSQL (Optional for local, defaults to SQLite)

### 1. Clone & Setup
```bash
git clone https://github.com/Yashuu213/Hisaab.AI
cd upi
```

### 2. Backend (Flask) Setup
```bash
# Open a terminal and run:
pip install -r requirements.txt
python api/index.py
```

### 3. Frontend (React) Setup
```bash
# Open a second terminal and run:
npm install
npm run dev
```

Your app will be live at `http://localhost:5173`. Boom. 💥
